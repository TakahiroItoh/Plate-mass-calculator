# Plate Mass Calculator

A lightweight static web app for calculating the mass of a rectangular plate from material density and dimensions.

## Features

- Material dropdown with density shown in `g/cm3`
- Inputs for width, length, and height in `mm`
- Mass output in `kg`
- Live isometric wireframe preview
- Plain HTML, CSS, and JavaScript with no build step

## GitHub Pages

This repository is structured for GitHub Pages deployment directly from the `main` branch root.
